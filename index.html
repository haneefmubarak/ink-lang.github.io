What is Jayfor?

Jayfor is a compiled programming language implemented in C with an LLVM backend. We created Jayfor because we love C, but there are some flaws that we want to address:
  • No built-in boolean type
  • Dangling Pointers
  • No strings
  • Inconsistent Error Handling
We hope to eliminate these problems in Jayfor. We aren't necessarily aiming at security at a Rust level, but we are going for a more user friendly language than C, something that makes sense.

Jayfor's Goals

  • Speed — We'd like a fairly fast language
  • Simplicity — We want it to be simple, and make sense
  • Cleaner syntax — It should roll off the tongue
  • Experiment — We want to experiment with new ideas

How it looks

use stdio.j4;

fn main(): int {
	println("hello, world.");
	return 0;
}

Status

  • Jayfor is still in development
  • The compiler is written in C
  • LLVM is used for the backend
